# 🍔 CrunchyFood – Smart Food Ordering System

A modern **full-stack Food Ordering Web Application** built with **Flask** and **MySQL**, designed to provide a seamless online food ordering experience for both customers and restaurant administrators.

The platform enables customers to browse menus, search for food items, manage their carts, and place orders, while providing administrators with a dedicated dashboard to manage menu items and monitor sales.

---

## 📌 Overview

CrunchyFood is a responsive restaurant management and food ordering system developed using **Python Flask**, **MySQL**, **HTML**, **CSS**, and **JavaScript**. The application follows a clean client-server architecture with secure authentication and role-based access control.

### Key Highlights

* Secure user authentication
* Customer & Admin role management
* Interactive shopping cart
* Order management system
* Admin dashboard for menu management
* Sales analytics dashboard
* Responsive UI for desktop and mobile devices

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Devansh-Mankad/CrunchyFood-Smart-Food-Ordering-System.git
```

---

## 2️⃣ Navigate to the Project Directory

```bash
cd CrunchyFood-Smart-Food-Ordering-System
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Configure MySQL

Create a MySQL database and update the database connection details in **app.py** according to your local environment.

---

## 5️⃣ Run the Application

```bash
python app.py
```

---

## 6️⃣ Open in Browser

```
http://127.0.0.1:5000
```
