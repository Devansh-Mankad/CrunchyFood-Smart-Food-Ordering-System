# CrunchyFood-Smart-Food-Ordering-System
Crunchy Food is a modern Food Ordering Web Application built using Python Flask, MySQL, HTML, CSS, and JavaScript. It provides a complete online food ordering experience with separate interfaces for customers and administrators.

The application allows users to browse menu items, search for food, add products to their cart, place orders, and manage their accounts. The admin panel enables restaurant staff to manage menu items by adding, editing, and updating food details through an intuitive dashboard.

🚀 Installation
1. Clone the repository
  git clone https://github.com/your-username/Crunchy_Food.git
2. Move into the project
  cd Crunchy_Food
3. Install dependencies
  pip install -r requirements.txt
4. Configure your MySQL database.
5. Run the Flask application
  python app.py
6.Open your browser
  http://127.0.0.1:5000
